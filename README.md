# dotfiles

## requirements
- language servers
  - pylsp
  - clangd

## setup
- install Jetbrains Mono Nerdfont
- run `install.sh`
- start tmux
  - install plugins with `Ctrl+Space+I`
- start nvim
  - run `:PackerSync` if not done automatically
